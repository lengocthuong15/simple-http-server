System Overview

  • 1 main thread for starting the server and for user interact
  • 1 Listenser thread to accept incoming from clients and then propagte the request to the workers
  • N worker threads (by default, it is 5): those threads take the requests from the Listenser and process HTTP requests and then sends the responses back to client.
